SP1: Spoopy Sprint
Group: Team Protag
This was our first game project at TGA, and this game was made
using Unity.
We were tasked with making an endless runner.
In this project I worked on a camera system.
It went unused however, but I learned a lot this project all the same.
I also rigged and animated a dancing skeleton that’s scattered across
the levels.

SP2: The Tale of Nether Oakfield
Group: Team Idle Rats
This time we were tasked with making a puzzle game for mobile devices, using
Unity yet again.
I worked on a cloud shader, I did some QA testing and I worked on performance and
stability fixes for mobile devices along with my team member Lukas Trossvik.

SP3: Mafia Man 2
Group: Datorspel
This was our first project in TGA’s internal engine TGE.
We were tasked with making a 2D-platformer.
In this project I made a particle system which we used for the player and for coins.

SP4: Stella’s Quest
Group: Team SP4GHETTI
This was our second project made in TGE.
This time I ported over the particle system from SP3 and added billboarding for the emitter’s sprites.
I also worked on a few assorted things like hunting memory leaks, bugfixing and so on.

SP5: A Brush With Death
Group: Team CatNap
This was the start of the second year, and this game was made using our
own engine called Kitty Engine.
We developed both Kitty Engine and this game at the same time over a short time span of 5 weeks.
The game was more of a tech demo than anything, because we had so little time and had to focus on developing a game engine.
Still, we were tasked with making a simple and short 3D-puzzle game.
I worked mostly on the sound system of the game, which meant things like
creating a wrapper around XAudio2, writing audio components and establishing their pipeline, handling playback with looping and so on.

SP6: Spite: Hymn of Hate
Group: Team CatNap
This was the second project developed in Kitty Engine.
Here I worked more on the sound system, and refactored it a lot,
making it easier to work with.
This time my work was more centered around polishing
the audio component pipeline.
I also worked on the animation system along with my
teammate Anton Eriksson.

SP7: Harbour Havoc
Group: Team CatNap
This was the third project developed in Kitty Engine.
Here I worked on the audio system yet again.
This time I integrated spatial audio and remodeled audio components
to allow for better audio concurrency, which meant being able to play back
the same audio component through new sets of voices at the same time, up to
a hard-coded limit.
